# Поиск инсайтов и точек роста

**Задачи проекта:** Провести комплексный анализ данных о поведении пользователей и эффективности партнеров-ресторанов для выявления ключевых точек роста в сервисе доставки еды.

**Описание проекта:** Выполнена предобработка данных, анализ воронки пользовательского пути и выявлено основное узкое место — низкая конверсия подтверждения телефона в мобильном приложении (58%). Проведена RFM-сегментация пользователей, которая показала преобладание одноразовых клиентов (более 70%) и низкую долю лояльных. С помощью кластеризации K-Means выявлены три типа ресторанов-партнёров: массовые, премиальные и малые/нишевые. На основе анализа подготовлены стратегические рекомендации по оптимизации регистрации, повышению удержания и дифференцированной работе с партнёрами.
